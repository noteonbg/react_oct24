Understanding CSS Units

100% Width

The above box uses 100% width, taking the full width of its parent container.

This text size is 1.5em (1.5 times the parent font size).

The box above is sized using em units, which are relative to the font size of the parent element.

This text size is 1.25rem (1.25 times the root font size).

Using rem units ensures the size is relative to the root element, making it more consistent across different parts of the document.

This box is 50vw wide (50% of the viewport width).

The box above uses vw, which allows it to adjust based on the width of the viewport.

This box is 50vh tall (50% of the viewport height).

The box above uses vh, which allows it to adjust based on the height of the viewport.
