npm install -g create-react-app

npx create-react-app my-app

This command starts the development server, and your app should open automatically in your default web browser, usually at http://localhost:3000.

Basic Components
src/index.js: The entry point of your application. It renders the App component.
src/App.js: The main component of your application. You can modify this file to start building your app.
public/index.html: The HTML template where your React app will be injected
