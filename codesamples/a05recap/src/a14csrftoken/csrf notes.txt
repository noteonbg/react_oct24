CSRF
CSRF Tokens: Require a unique token for each state-changing request (e.g., submitting forms). The server generates this token, and the client must include it in requests

SameSite Cookies: Use the SameSite attribute for session cookies to prevent them from being sent with cross-origin requests.

Referer Header Validation: Check the Referer header to ensure that the request is coming from a trusted origin.
